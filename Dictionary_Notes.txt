Four syntaxes -- dict 
**********************

# Syntax 1
# How to fetch the value if you know the key 
 print(colors["brown"])

# Syntax 2
# How to add a new element into the dict 
 colors["dark blue"] = 36

# Syntax 3
# How to modify the value of an existing element in a dict.
 colors["brown"] = 88

# Sytax 4
# How to check whether a key already exists in the dict.

 "red" in colors
